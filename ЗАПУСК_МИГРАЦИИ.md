# Запуск миграции данных в PostgreSQL

## Способ 1: Через PowerShell (рекомендуется)

Откройте PowerShell и выполните:

```powershell
cd C:\Users\Driga_VA\Academy_SetlSoft\ID\backend
node scripts/migrate-local-data.js
```

## Способ 2: Через двойной клик

1. Откройте папку `ID\backend`
2. Дважды кликните на файл `migrate.bat`

## Способ 3: Если нужно установить зависимости

Если получите ошибку "Cannot find module 'pg'", сначала установите зависимости:

```powershell
cd C:\Users\Driga_VA\Academy_SetlSoft\ID\backend
npm install
node scripts/migrate-local-data.js
```

---

**Файл `migrate-local-data.js` уже содержит DATABASE_URL, копировать ничего не нужно!**

После миграции просто обновите страницу на продакшене.
